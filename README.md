# topcallerstockbot

Telegram бот для прогнозирования цен акций с ML/AI.

Функции

- Анализ акций: AAPL, TSLA, MSFT, GOOGL...
- 3 ML модели: Ridge Regression, ARIMA, LSTM
- Прогноз на 30 дней + график
- BUY/SELL сигналы + расчет прибыли

This bot is for educational purposes only. Not financial advice.
